# 番茄象限 — Tomato Matrix

🍅 番茄工作法 × 四象限法则 = 高效日常任务管理工具

## 功能

- **四象限面板**：紧急/重要二维矩阵，拖拽排序
- **番茄钟**：自定义时长，暂停/继续，休息自动恢复
- **日历选择**：月视图，日期切换
- **月度报表**：TXT/CSV 导出
- **数据持久化**：localStorage 存储，离线可用
- **可配置**：专注时长、休息时长、每日目标

## 技术栈

React + TypeScript + Vite + Tailwind CSS + Zustand + @dnd-kit

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
